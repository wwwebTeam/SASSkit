SASSkit is a starter kit for your next website. It's completely based on SASS, but jQuery is included as well.

The SASS layout explained

The SASS is very organized in the assets/css/ folder. All the files are imported into the file /assets/css/app.sass, that file gets outputted by sass into /assets/css/app.css

There are many files in /assets/css/, but worry not, because everything is nicely organized. In the /assets/css/1-tools/ folder, all the tools are placed.

  • bourbon
  • fonts.scss
  • normalize.scss
  • vars.sass

Then there is the /assets/css/2-basics/ folder. In that folder are all the stylesheets that make the site a nicer site. It adds font, selection colors nice margins and paddings and much more. This is what's included:

  • body-elements.sass
  • link.sass
  • selection-colors.sass
  • typography.sass

In the /assets/css/3-modules/ folder, you can add your own modules. It's not something basic, but you do want to have it available on all your pages, than is this the folder to use. There are some example files included:

  • EXAMPLE-ctas.sass
  • EXAMPLE-figure.sass

Last, but not least is the /assets/css/4-pages/ folder. You can add all the files for your specific pages. There's already added an example file for this page, but you can make as many pages as you want.

  • EXAMPLE-page-home.sass

The JS explained

In the /assets/js/ folder, there are two files in total: the /assets/js/jquery-3.2.1.min.js file and the /assets/js/app.js file. The first one is the latest jQuery. The other one is the main javascript file where you're going to write all your functions in. Here's the layout one more time:

  • jquery-3.2.1.min.js
  • app.js
